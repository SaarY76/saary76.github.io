How To download and install View file

First Download in the previous page the View file

Move the Views in the downloaded Zip file to the folder that shows in the picture, usually it's in :
C:\Users\Your User Name\Documents\Sports Interactive\Football Manager 2024\views

Now open the Football Manager game

For this tutorial I will load it to the Players search table but it's also the same for the Shortlist and the Squad tables

In the left menu of the screen go to Scouting And then in the top menu click on Players

To import the View to the Player Serach table click on the button that is right to the text that show how many players found and follow the next steps in the next pictures


When you see this screen popping click on the FM All Attributes file you downloaded
and then click on the Load button

Now you will see the View with all of the new columns in the players search table

These are also the instructions for the shortlist table and the squad screens to load up their Views
