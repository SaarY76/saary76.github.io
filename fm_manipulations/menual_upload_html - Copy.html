How to Upload your HTML Players file

First go to your Players Search screen in the Football Manager game

Filter the results of the amount of players to MAXIMUM of 4534 players

Click on one of the lines that represent player in the a players table and then click : CTRL A
It should look like this :

After selecting all of the players click : CTRL P
And it will show you a popping scrren like this :

Select the Web Page option and then click OK

Now it will show you a popping screen that let you choose where do you want to store the HTML file in you PC as you see in the next picture. Put the file in a place you will remember and then click : Save

Now you saved your HTML file with the players data and ready to upload it !
